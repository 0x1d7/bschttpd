Welcome to Basic Httpd

If you see this page, your Basic Httpd web server is successfully installed and working.

Thank you for using Basic Httpd. We hope you enjoy your web serving experience!

Features

  • Static File Handling: Efficiently serves static files with tools to customize configurations.
  • In-Memory Caching: Supports caching mechanisms to enhance performance and reduce server load.
  • Asynchronous I/O: Fully utilizes asynchronous I/O operations for optimal performance.
  • Native AOT: Compiled to native code for high performance and minimal footprint.

Configuration

Basic Httpd can be easily configured through the appsettings.conf file:

        {
          "WebServerConfiguration": {
            "HttpsRedirection": false,
            "Wwwroot": "/srv/wwwroot",
            "DefaultDocument": "index.html"
          },
          "Kestrel": {
            "Endpoints": {
              "Http": {
                "Url": "http://*:8080",
                "Protocols": "Http1"
              },
              "Https": {
                "Url": "https://localhost:8443",
                "Protocols": "Http2",
                "Certificate": {
                  "Path": "/etc/ssl/certs/localhost.pem",
                  "KeyPath": "/etc/ssl/private/localhost.key"
                }
              }
            }
          }
        }

You can change the default page by adjusting the DefaultDocument property in the configuration file.

Endpoints can also be configured to specify HTTP and HTTPS protocols, ports, and certificates.
